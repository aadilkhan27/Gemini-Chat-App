## What makes this different from basic Gemini tutorials
- 🔴 **Live token streaming** — renders each word as Gemini generates it
- ⏹️ **Stop generation** — cancel mid-response with a single tap
- 📝 **Markdown + code blocks** — properly formatted AI responses
- 📴 **On-device AI ready** — architecture supports ML Kit / Gemini Nano bridge
- 🌙 **Dark mode** — full Material 3 theming
